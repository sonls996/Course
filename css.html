KHÓA HỌC CSS CĂN BẢN

"CSS là gì" là câu hỏi muôn thưở của những bạn mới bắt đầu học CSS. Về đinh nghĩa thì nó không phức tạp lắm nhưng về cách sử dụng thì có lẽ là một chặng đường dài và đầy gian nan. Chính vì vậy mình muốn tạo một series học CSScăn bản để các bạn có một nơi học chất lượng vã đặc biệt là miễn phí 100%. Trước tiên chúng ta sẽ tìm hiểu định nghĩa trước rồi sau đó tìm hiểu cách học nó nhé.

1.CSS là gì ?

CSS là chữ viết tắt của cụm từ " Cascading Style Sheet", dùng để trình bày bố cục của một Website dựa trên các thẻ HTML. Nói nôm na thì CSS sẽ tác động thay đổi cách hiển thị mặc định của các thẻ HTML nhằm thân thiện với người dùng hơn. CSS đóng vai trò rất quan trọng khi bạn xây dựng giao diện Website. Thông thường bạn sẽ sử dụng một chương trình thiết kế như Photoshop để tạo giao diện, sau đó các Front-end Developer sẽ dựa vào file thiết kế đó kết hợp với HTML & CSS để convert thành một file có đuôi mở rộng là .html, file này sẽ chưa code HTML & CSS.

Có một ví dụ trực quan để diễn tả nhiệm vụ của CSS như sau, giả sử bạn xây nhà và bạn sẽ sử dụng gạch để xây dựng ngôi nhà. Nhưng để trang trí cho ngôi nhà thì bạn sẽ sử dụng một số đồ nội thất khác như sơn màu, ... thì lúc này sơn màu sẽ tác động làm thay đổi color mặc định của gạch. Nói ví von thi màu sơn chính là CSS và gạch chính là HTML.

Gom nhóm các thể loại của CSS

CSS có nhiều kiểu nhưng gom lại thì chúng ta sẽ có các thể loại CSS như sau :

  • Background : CSS tùy chỉnh hình nền
  • Text : CSS tùy chỉnh cách hiển thị của đoạn text
  • Font : CSS tùy chỉnh kích thước, kiểu chữ
  • Link : CSS tùy chỉnh link
  • List : CSS tùy chỉnh link
  • Table : CSS tùy chỉnh link
  • Box model : Mô hình box model kết hợp padding, margin, border
  • ....
2. Cú pháp CSS, các cách viết CSS

Chúng ta vừa mới tìm hiểu được khái niệm CSS là gì rồi. Vậy thì trong phần này để cho nóng thì ta sẽ tìm hiểu đến cú pháp viết CSS, đồng thời tìm hiểu ba cách trình bày CSS ( inline, external, internal )

2.1 Cú pháp CSS

Chúng ta chia một đoạn mã CSS thành 2 phần chính :

  • selector:selector sẽ trỏ đến những đối tượng HTML chịu ảnh hưởng bởi CSS
  • declaration: Các thuộc tính dùng để style cho thẻ selector

Ví dụ: Thiết lập màu nền và chữ màu đỏ cho thẻ H1

1    h1 {

2       background  : red ;

3       color  : red ;

4    }

Trong đó :

  • h1 là selector
  • Các thuộc tính background và color chính là declaration

Riêng về selector thì mình sẽ trình bày ở phần tiếp theo.

Còn đối với declaration thì các bạn để ý bên trong cặp ngoặc {  } là danh sách các thuộc tính và giá trị tương ứng theo dạng ( Property:Property value )

mỗi thuộc tính sẽ kết thuộc bằng dấu chấm phẩy ( ; ).
2.2 Các cách viết CSS

Ở phần Cú pháp CSS có lẽ vẫn còn hơi mờ hồ đúng không nào? Vậy thì phần này sẽ thực sự thú vị hẳn đấy vì chúng ta sẽ tìm hiểu các cách viết CSS và những lưu ý khi sử dụng chúng.

Chúng ta có ba cách viết CSS đó là :

  • inline: Viết trực tiếp trên thẻ thông qua thuộc tính style
  • external: Viết riêng một file có đuôi dạng .css rồi sau đó import vào file HTML thông qua thẻ link
  • internal: Viết trực tiếp tại file HTML hiện tại thông qua thẻ style

#INLINE

Trong thẻ HTML chúng ta tạo một thuộc tính style="code css".

1 < div style = "background : red; color : blue ;" > Cùng học lập trình miễn phí tại Super Code < /div >

Không nên sử dụng phương pháp inline vì nó khiến ta khó quản lý code và không tốt cho SEO

#INTERNAL

Chúng ta sẽ code bên trong thẻ < style type="test/css"> Viết code ở đây < /style >

1< style type= "text/css" >

2  div {

3    background :red;

4    color : blue ;

5     }

7 </style>

8 < div> HỌC CSS MIỄN PHÍ TẠI SUPER CODE < /div>

Dạng internal có thể chấp nhận được nhưng bạn nên đặt CSS ở trên thẻ HEAD, dạng này cũng không tốt cho SEO

#EXTERNAL

Cách này hơi rườm rà một chút vì chúng ta cần 1 file style.css ( có phần mở rộng là .css ) và sau đó import vào file HTML thông qua thẻ link. Sau đây là một ví dụ cho cách này.

Bước 1 : Tạo 1 file style.css với nội dung như sau :

1  div {

2    background : red ;

3     color : blue;

4    }

Bước 2 : Tạo 1 file index.html cùng cấp với file style.css với nội dung như sau :

1  < link href= "style.css" rel= "stylesheet"/>

2  <div> HỌC LẬP TRÌNH MIỄN PHÍ TẠI SUPER CODE </div>

Sau đó chạy lên các bạn sẽ thấy kết quả cũng không có gì khác so với hai ví dụ trên. Nhưng có một số lưu ý dành cho bạn như sau:

  • Trong thẻ link có một thẻ href, bạn sẽ truyền đường dẫn đến file css của bạn.
  • rel="stylesheet" báo cáo cho trình duyệt biết đây là file CSS.
3. Selector là gì? Tìm hiểu Selector cơ bản?

CSS Selector đóng vai trò rất quan trọng khi các bạn code layout cho website, tuy nhiên về các loại selector thì hơi nhiều nên trong phần này tôi chỉ trình bày một số selector thông dụng và căn bản nhất và hay sử dụng nhất, các vấn đề nâng cao mình sẽ trình bày trong khóa học css nâng cao.

1. Selector là gì ?

Selector nếu dịch ra tiếng anh thì có nghĩa là " người chọn", nhưng trong CSS thì selector dùng để truy vấn đến các thẻ HTML.

Như các bạn biết trong 1 file HTML thì có rất nhiều thẻ giống nhau và thông thường chúng ta sẽ đặt các ID,Class cho các thẻ để phân biệt, vì vậy CSS sẽ dựa vào các ID,Class để truy xuất tới các thẻ HTML và người ta gọi đó là selector.

Ví dụ : Truy xuất tới tất cả các thẻ DIV và gán background cho nó màu đỏ

1  div{

2  background : red;

3  }

Quá đơn giản phải không nào. Bây giờ ta sẽ tìm hiểu những loại selector cụ thể thông dụng trong CSS

Các CSS selector thông dụng

Có rất nhiều loại selector nhưng trong bài này chúng ta tìm hiểu một vài loại selector hay dùng. Trong bài này chúng ta chỉ tìm hiểu hai cách chính đó là:

  • CSS Selector ID
  • CSS Selector Class

Selector ID

Bạn lưu ý là trong một trang web ID là duy nhất nhé, nghĩa là nếu bạn định nghĩa hai ID giống nhau trong 1 layout thì không đúng chuẩn giao diện của W3C.

Giả sử bạn có nhiều thẻ div và bạn muốn viết CSS cho một thẻ DIV nào đó thôi thì bạn có thể chọn giải pháp là Selector theo ID của HTML. Chúng ta sử dụng dấu thăng (# ) để đại diện cho ID .

Ví dụ: Cho background màu đỏ cho div có id ="active"

HTML

1   <div id="active">

2       ACTIVE

3   </div>

4   <div>

5      NON-ACTIVE

6   </div>

CSS

1  #active {

2   background : red ;

3  }

Rất là dễ hiểu đúng không nào, tiếp theo chúng ta sẽ tìm hiểu sang loại selector thứ hai nhé.

Selector Class

Với ID là duy nhất thì Class ngược lại, nghĩa là bạn có thể cho nhiều thẻ HTML có cùng tên Class, điều này khá tiện lợi cho CSS. Ví dụ bạn cần style cho một số thẻ DIV nào đó thôi thì nếu bạn dùng ID thì không hay lắm vì phải viết nhiều lần, chính vì vậy ta sẽ chọn class. Selector cho class sẽ là dấu chấm (.)

Ví dụ: Cho background màu xanh cho div có class ="active"

HTML

1   <div class="active">

2       ACTIVE

3   </div>

4   <div>

5      NON-ACTIVE

6   </div>

CSS

1  .active {

2   background : blue ;

3  }

Còn các selector nâng cao nhưng mình nghĩ sẽ giới thiệu ở một bài khác vì nếu đưa vào đây luôn thì sẽ hơi rối cho những bạn mới học, khi nao chúng ta nhuần nhuyễn rồi thì học tiếp cũng chưa muộn. Nhưng chung quy lại nếu bạn thuộc các selector căn bản trên là có thể cắt HTMl được rồi đó. Hy vọng qua bài này bạn sẽ hiểu được selector là gì.

4. Thiết lập màu nền với background

Trong phần này chúng ta sẽ tìm hiểu cách thiết lập màu nền cho các thẻ HTML, đây là một thuộc tính khá quan trọng mà bạn cần phải nắm vững vì hầu hết các trang web đều có sử dụng thuộc tính này để tạo background với các màu sắc khác nhau hoặc hình ảnh.

CSS thiết lập màu nền cho background

Để thiết lập màu nền cho background thì ta sử dụng thuộc tính CSS backgroud-color hoặc background với giá trị của nó là màu sắc của background. Bạn có thể sử dụng mã màu hoặc tên màu bằng tiếng anh đều được.

Ví dụ : Thiết lập background cho toàn trang màu đỏ

Thẻ mà ta có thể thiết lập background cho toàn trang là thẻ body nên ta chỉ cần gán CSS cho nó là được

1    body {

2      background : red ;

3   }

Hoặc

1    body {

2      background : #fb0320 ;

3   }

Hoặc

1    body {

2      background-color : #fb0320 ;

3   }

Trong bài này chủ yếu tìm hiểu thuộc tính CSS xử lý background, nhưng vẫn còn nhiều thuộc tính nữa nhưng không trình bày nó trong bài này, bạn phải học từ từ mới thấm trong người được chứ vội quá quên hết. Chúc các ban học CSS vui vẻ :D

5. Các thuộc tính với định dạng text

CSS cung cấp cho chúng ta một số thuộc tính định dạng text như màu sắc, hiển thị chữ in hoa hoặc in thường, vị trí hiển thị, ... Những thuộc tính này khá hay và rất quan trọng vì nó quyết định tính thẩm mỹ cho các dòng chữ trong website.

Trong bài chúng ta tìm hiểu một số thuộc tính định dạng cho text như sau:

  • color
  • text-transform

    • - Chọn màu cho chữ với thuộc tính Color

    Ta sử dụng thuộc tính color để gán màu sắc cho chữ

    Trong CSS thì ta sử dụng 3 cách biểu diễn color như sau:

    • HEX : có định dạng #mã-màu
    • RGB : có định dạng rgb(xxx,yyy,zzz)
    • Tên màu : ví dụ red, blue
    Ví dụ : Thiết lập màu chữ cho ba class color--red, color--blue, color--pink

    1.color--red {

    2    color : red ;

    3  }

    4.color--blue {

    5   color : #2980b9 ;

    6  }

    7.color--pink {

    8   color : rgb rgb(255, 192, 203);

    9  }

    - Thiết lập chữ in hoa mà in thường

    Nếu bạn muốn một thẻ nào đó luôn luôn hiển thị in hoa hoặc luôn luôn hiển thị in thường mặc dù ta nhập vào là hoa hay thường thì sử dụng thuộc tính text-transform với các thuộc tính:

    • uppercase: chuyển đổi in hoa
    • lowercase: chuyển đổi in thường
    • none: không chuyển đổi gì cả
    Ví dụ : Dùng kiến thức vừa học, chuyển đổi chữ thành chữ in hoa và in thường

    1 .uppercase {

    2     text-transform : uppercase ;

    3  }

    4 .lowercase {

    5    text-transform : lowercase ;

    6  }

    7 .none {

    8    text-transform : none ;

    9  }
