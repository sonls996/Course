1.CSS là gì ?

CSS là chữ viết tắt của cụm từ " Cascading Style Sheet", dùng để trình bày bố cục của một Website dựa trên các thẻ HTML. Nói nôm na thì CSS sẽ tác động thay đổi cách hiển thị mặc định của các thẻ HTML nhằm thân thiện với người dùng hơn. CSS đóng vai trò rất quan trọng khi bạn xây dựng giao diện Website. Thông thường bạn sẽ sử dụng một chương trình thiết kế như Photoshop để tạo giao diện, sau đó các Front-end Developer sẽ dựa vào file thiết kế đó kết hợp với HTML & CSS để convert thành một file có đuôi mở rộng là .html, file này sẽ chưa code HTML & CSS.

Gom nhóm các thể loại của CSS

CSS có nhiều kiểu nhưng gom lại thì chúng ta sẽ có các thể loại CSS như sau :

  • Background : CSS tùy chỉnh hình nền
  • Text : CSS tùy chỉnh cách hiển thị của đoạn text
  • Font : CSS tùy chỉnh kích thước, kiểu chữ
  • Link : CSS tùy chỉnh link
  • List : CSS tùy chỉnh link
  • Table : CSS tùy chỉnh link
  • Box model : Mô hình box model kết hợp padding, margin, border
  • ....
2. Cú pháp CSS, các cách viết CSS

Chúng ta vừa mới tìm hiểu được khái niệm CSS là gì rồi. Vậy thì trong phần này để cho nóng thì ta sẽ tìm hiểu đến cú pháp viết CSS, đồng thời tìm hiểu ba cách trình bày CSS ( inline, external, internal )

2.1 Cú pháp CSS

Chúng ta chia một đoạn mã CSS thành 2 phần chính :

  • selector:selector sẽ trỏ đến những đối tượng HTML chịu ảnh hưởng bởi CSS
  • declaration: Các thuộc tính dùng để style cho thẻ selector

Ví dụ: Thiết lập màu nền và chữ màu đỏ cho thẻ H1

1    h1 {

2       background  : red ;

3       color  : red ;

4    }

Trong đó :

  • h1 là selector
  • Các thuộc tính background và color chính là declaration

Riêng về selector thì mình sẽ trình bày ở phần tiếp theo.

Còn đối với declaration thì các bạn để ý bên trong cặp ngoặc {  } là danh sách các thuộc tính và giá trị tương ứng theo dạng ( Property:Property value )

mỗi thuộc tính sẽ kết thuộc bằng dấu chấm phẩy ( ; ).
2.2 Các cách viết CSS

Ở phần Cú pháp CSS có lẽ vẫn còn hơi mờ hồ đúng không nào? Vậy thì phần này sẽ thực sự thú vị hẳn đấy vì chúng ta sẽ tìm hiểu các cách viết CSS và những lưu ý khi sử dụng chúng.

Chúng ta có ba cách viết CSS đó là :

  • inline: Viết trực tiếp trên thẻ thông qua thuộc tính style
  • external: Viết riêng một file có đuôi dạng .css rồi sau đó import vào file HTML thông qua thẻ link
  • internal: Viết trực tiếp tại file HTML hiện tại thông qua thẻ style

#INLINE

Trong thẻ HTML chúng ta tạo một thuộc tính style="code css".

1 < div style = "background : red; color : blue ;" > Cùng học lập trình miễn phí tại Super Code < /div >

Không nên sử dụng phương pháp inline vì nó khiến ta khó quản lý code và không tốt cho SEO

#INTERNAL

Chúng ta sẽ code bên trong thẻ < style type="test/css"> Viết code ở đây < /style >

1< style type= "text/css" >

2  div {

3    background :red;

4    color : blue ;

5     }

7 </style>

8 < div> HỌC CSS MIỄN PHÍ TẠI SUPER CODE < /div>

Dạng internal có thể chấp nhận được nhưng bạn nên đặt CSS ở trên thẻ HEAD, dạng này cũng không tốt cho SEO

#EXTERNAL

Cách này hơi rườm rà một chút vì chúng ta cần 1 file style.css ( có phần mở rộng là .css ) và sau đó import vào file HTML thông qua thẻ link. Sau đây là một ví dụ cho cách này.

Bước 1 : Tạo 1 file style.css với nội dung như sau :

1  div {

2    background : red ;

3     color : blue;

4    }

Bước 2 : Tạo 1 file index.html cùng cấp với file style.css với nội dung như sau :

1< link href= "style.css" rel= "stylesheet"/>

2<div> HỌC LẬP TRÌNH MIỄN PHÍ TẠI SUPER CODE </div>

Sau đó chạy lên các bạn sẽ thấy kết quả cũng không có gì khác so với hai ví dụ trên. Nhưng có một số lưu ý dành cho bạn như sau:

  • Trong thẻ link có một thẻ href, bạn sẽ truyền đường dẫn đến file css của bạn.
  • rel="stylesheet" báo cáo cho trình duyệt biết đây là file CSS.
3. Selector là gì? Tìm hiểu Selector cơ bản?

CSS Selector đóng vai trò rất quan trọng khi các bạn code layout cho website, tuy nhiên về các loại selector thì hơi nhiều nên trong phần này tôi chỉ trình bày một số selector thông dụng và căn bản nhất và hay sử dụng nhất, các vấn đề nâng cao mình sẽ trình bày trong khóa học css nâng cao.
