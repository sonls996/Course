"Javascript là gì" là câu hỏi muôn thưở của những bạn mới bắt đầu học . Trong loạt series này chúng ta sẽ tìm hiểu ngôn ngữ Javascript căn bản giúp các bạn newbie hoặc những bạn đã có kiến thức Javascript rồi nhưng muốn học để bổ trợ thêm, nắm và hiểu rõ được các khái niệm căn bản của javascript. Vì dạng viết text nên nội dung sẽ hơi khó diễn tả và khó trình bày nên các bạn cố gắng đọc từng chữ và thực hành theo demo nhé.

1. Javascript là gì ?

Javascript là một ngôn ngữ lập trình kịch bản dựa vào đối tượng phát triển có sẵn hoặc tự định nghĩa ra, javascript được sử dụng rộng rãi trong các ứng dụng Website. Javascript được hỗ trợ hầu như trên tất cả các trình duyệt như Firefox, Chrome, ... thậm chí các trình duyệt trên thiết bị di động cũng có hỗ trợ.

Nếu bạn đã biết qua về HTML5 thì bạn thấy có các khái niệm như sessionStore hay localStore, đây là hai đối tượng được tạo nên từ Javascript nên rõ ràng trong HTML5 cũng có sử dụng nó. Với những ứng dụng đó thì bạn thấy javascript không thể chết trong các ứng dụng website được.

JavaScript (viết tắt là "js") là một ngôn ngữ lập trình mang đầy đủ tính năng của một ngôn ngữ lập trình động mà khi nó được áp dụng vào một tài liệu HTML, nó có thể đem lại khả năng tương tác động trên các trang web. Cha đẻ của ngôn ngữ này là Brendan Eich. JavaScript thật sự rất linh hoạt. Bạn có thể bắt đầu với các bước nhỏ, với ? viện ảnh, bố cục có tính thay đổi và phản hồi đến các nút nhấn. Khi có nhiều kinh nghiệm hơn, bạn có thể tạo ra các trò chơi, hoạt họa 2 chiều hoặc 3 chiều, ứng dụng cơ sở dữ liệu toàn diện và nhiều thứ khác!

2. Cú pháp, các cách viết Javascript

Bước đầu tiên để học Javascript là bạn phải chuẩn bị một phần mềm để viết mã Javascript, mình thì thích dùng Visual Studio Code nhưng nếu bạn là người mới thì nên sử dụng Notepad++ vì nó rất nhẹ và dễ cài đặt.

Qua khái niệm javascript là gì? thì bạn thấy tất cả các trình duyệt đều hỗ trợ nên ta không cần phải download thư viện gì cả mà thông qua những đoạn mã script giúp trình duyệt nhận diện được đó là Javascript.

Cặp thẻ mở và đóng.

Tất cả những đoạn mã Javascript đều phải đặt trong cặp thẻ mở < script > và thẻ đóng < /script >, ví dụ:

Mã JavaScript cụ thể được nhúng vào trong tài liệu HTML bằng thẻ script. Chúng ta có thể nhúng nhiều script vào trong cùng một tài liệu, mỗi script nằm trong một thẻ SCRIPT. Khi trình duyệt gặp phải một thẻ script nào đó, nó sẽ đọc từng dòng một cho đến khi gặp thẻ đóng script. Tiếp đến nó sẽ kiểm tra lỗi trong các câu lệnh JavaScript. Nếu gặp phải lỗi, nó sẽ cho hiển thị lỗi đó trong chuỗi các hộp cảnh báo lên màn hình. Nếu không có lỗi, các câu lệnh sẽ được biên dịch sao cho máy tính có thể hiểu được lệnh đó.

Cặp thẻ script

1 < script language = "javascript">

2      alert ( "Hello World ") ;

3  < /script>

Đặt thẻ script ở đâu ?

Chúng ta có ba cách thường được sử dụng sau đây :

Cách 1 : Internal - viết trong file HTMl hiện tại

Thông thường chúng ta sẽ viết những đoạn mã javascript trên phần head, tuy nhiên đó không phải là điều kiện bắt buộc, nghĩa là bạn có thể đặt ở đâu tùy thích miễn là những đoạn mã đó phải được bao lại bằng thẻ script.

Ví dụ đặt trong thẻ head

1  <html>

2          <head>

3              <title> Ví dụ đặt trong thẻ head </title>

4             < script language = "javascript">

5              alert ( "Hello World ") ;

6              < /script>

7        < /head>

8      < body>

9             Somthing code here !

0      < /body>

1  < /html>

Ví dụ đặt trong thẻ body

1  <html>

2          <head>

3              <title> Ví dụ đặt trong thẻ head </title>

4        < /head>

5      < body>

6             < script language = "javascript">

7              alert ( "Hello World ") ;

8              < /script>

9      < /body>

0  < /html>

Cách 2 : External - viết ra một file js khác rồi import vào

Bạn có thể viết những đoạn mã javascript ở một file có phần mở rộng là .js, sau đó dùng thẻ script để import vào ( giống như đối với css vậy ). Ví dụ file JS của mình có tên là demo.js thì lúc này mình có thẻ import như sau :

Code import js

1 < script language =" javascript " src=" demo.js " > < /script>

Lúc này các đoạn mã bên trong file demo.js bạn không cần phải bọc nó trong thẻ script nữa, vì nó là file có đuôi là .js nên trình duyệt tự nhận diện đây là file chứa mã javascript

Cách 3 : Inline - viết trực tiếp trong thẻ HTML

Inline nghĩa là bạn sẽ viết những đoạn mã Javascript trực tiếp trong thẻ HTML luôn. Như ví dụ dưới đây là mình đang viết dạng inline vì đoạn mã alert(1) được đặt trong sự kiện onclick của thẻ button.

Code bên trong sự kiện click

1 < input type =" button " onclick=" alert(1) " value= "click me" />

3. Viết chương trình Hello World đầu tiên

Trước khi thực hành thì bạn phải chắc chắn là bạn đã cài đặt Notepad++ để viết mã Javascript nhé, hoặc bạn có thể sử dụng một phần mềm code bất kì.

Bước 1 : Bây giờ chúng ta thực hành nhé, bạn hãy tạo một file index.html và lưu tại bất kì vị trí nào, desktop hay ổ D, ổ C gì đó thì tùy bạn, miễn là phần mở rộng là .html, sau đó mở file đó bằng Notepad++ rồi gõ nội dung sau vào:

Mã HTML

1  <html>

2          <head>

3              <title> </title>

4        < /head>

6         <input type="button" value=" Click me" />

5      < body>

7      < /body>

8  < /html>

Chạy lên bằng Firefox hoặc Chrome bạn thấy xuất hiện một button. Bây giờ ta sẽ viết ứng dụng khi click vào button đó thì sẽ xuất hiện một thông báo "Hello World" nhé. Có lưu ý là button đó mình có đặt id = " clickme " nha.

Bước 2 : Viết đoạn mã Javascript khi click vào button có id = " clickme" thì sẽ có thông báo hiện lên màn hình

Toàn bộ chương trình

1  <html>

2          <head>

3              <title> Viết chương trình Hello World đầu tiên </title>

4        < /head>

5      < body>

6         <input type="button" value=" Click me" id= "clickme" />

7             < script language = "javascript">

8                  var button = document.getElementById("clickme");

9                   button.addEventListener(" click ",function() {

0                   alert (" Hello World ");

1                   } ) ;

2              < /script>

3      < /body>

4  < /html>

Thử xem

Câu lệnh : < input type = " button " id = "clickme" value = " Click me" /> : Tạo một nút xác định có thể nhấn với id="clickme"

Câu lệnh : var button = document.getElementById('clickme'); : Đây là cách để khai báo biến của javascript, biến button được gán giá trị sau dấu "=", button sẽ có giá trị là document.getElementById('clickme')

document.getElementById('clickme') là phương thức của javascript, nó sẽ truy xuất tới phần tử có id = "clickme", bước đầu để ta có thể thao tác lên nó.

Câu lệnh : button.addEventListener("click",function()) khi click vào phần tử chứa trong button thì sẽ thực hiện một function, addEventListener là một phương thức của javascript, nó đính kèm một trình sử lý sự kiện vào phần tử đã chỉ định, và sự kiện ở đây là người dùng click vào button có id = "clickme".

Câu lệnh : Alert(" Hello World); là một phương thức hiển thị một hộp cảnh báo với thông báo được chỉ định bên trong ngoặc nhọn () và nút OK. Nhằm đảm vào thông tin đến người dùng và ở đây nội dung cảnh báo là "Hello World" .

Lưu ý là bạn phải đặt đoạn mã javascript bên dưới button như trong demo nhé, nếu bạn đặt ở trên là sẽ bị lỗi đấy, lý do tại sao thì trong các bài tiếp theo mình sẽ giải thích kỹ hơn.

4. Các thư viện Javascript đình đám hiện nay

Những ứng dụng to lớn của Javascript khiến người ta không thể quên nó được. Hiện nay có rất nhiều thư viện và framework được viết từ Javascript như:

  • Angular JS: Một thư viện dùng để xây dựng ứng dụng Single Page
  • NodeJS: Một thư viện được phát triển phía Server dùng để xây dựng ứng dụng realtime
  • Sencha Touch: Một Framework dùng để xây dựng ứng dụng Mobile
  • ExtJS: Một Framework dùng xây dựng ứng dụng quản lý (Web Applications)
  • JqueryMột thư viện rất mạnh về hiểu ứng
  • ReactJSMột thư viện viết ứng dụng mobile
  • Và còn nhiều thư viện khác

Còn rất rất nhiều nhưng mình chỉ liệt kê bấy nhieu thôi nhé :D
